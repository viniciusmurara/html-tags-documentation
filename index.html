Welcome to HTML Tags Documentation!

That's a website designed to help you explore and master HTML tags with examples and explanations.

h1 / h2 / h3 / h4 / h5 / h6

The <h1> to <h6> tags define headings.

<h1> is the most important heading (largest), and <h6> is the least important (smallest).

p

The <p> tag defines a paragraph of text.

It is a block-level element and usually adds space before and after the text.

a

The <a> tag defines a hyperlink.

It is used to link to another page, file, location, or resource. The href attribute specifies the destination URL.

ul / ol / li

The <ul> tag defines an unordered list (bulleted).

The <ol> tag defines an ordered list (numbered).

The <li> tag defines each list item inside <ul> or <ol>.

img

The <img> tag embeds an image in the page.

The src attribute specifies the image path, and the alt attribute provides alternative text.

form / input / button

The <form> tag defines an interactive form to collect user input.

The <input> tag is used for fields like text boxes, checkboxes, or radio buttons.

The <button> tag defines a clickable button to submit or trigger actions.

table / tr / td / th

The <table> tag creates a table.

The <tr> tag defines a row, <td> defines a cell (data), and <th> defines a header cell.

header / main / section / article / footer

<header> defines the introductory content, often with logo or navigation.

<main> defines the central content of the page, unique and essential.

<section> groups related content within the page.

<article> represents an independent piece of content, like a blog post or news item.

<footer> defines the footer area, often with author info, links, or copyright.
